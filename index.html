USER GUIDE

  1. Copy Paste/drop file EJ Log sesuai periode hari replenish tanpa di potong
  2. Jika pada file EJ Log terdapat lebih dari satu periode maka App otomatis menganalisis periode terakhir
  3. Input data fisik remaining sesuai dengan hasil hitung fisik petugas CPC
  4. Jika terdapat kaset defect yang tidak terbaca pada saat replenish maka input manual jumlah total pagu replenish
  5. Click filter data
  6. Jika terjadi error atau hasil acak, potong EJ sesuai Periode replenish
  7. Aplikasi ini untuk membantu mempercepat penyelesaian tugas dan bukan untuk pengambilan keputusan, seluruh nilai yang di inut pada laporan akhir sepenuhnya di pertanggung jawabkan oleh managerial onduty atau terkait

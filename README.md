# REKONSILIATOR - Command Center v4.1 (Secure & Smart AI)

Aplikasi web untuk rekonsiliasi log mesin ATM dengan fitur AI yang canggih.

## Fitur Utama

- **Multi-Vendor Support**: Hyosung, Wincor, NCR, CRM Hitachi, dan ATM Jalin
- **Enhanced AI Assistant**: Analisis log dengan kemampuan AI yang ditingkatkan
- **Security Gate**: Sistem keamanan dengan login password
- **Drag & Drop**: Upload file log dengan mudah
- **Realtime Analysis**: Analisis dan rekonsiliasi data secara real-time
- **Responsive Design**: Tampilan yang optimal di berbagai perangkat

## Struktur Folder
Recon-Ejlog/
├── index.html # File utama HTML
├── style.css # Stylesheet kustom
├── script.js # JavaScript logic
├── assets/ # Folder untuk assets
│ ├── images/ # Gambar dan icon
│ └── fonts/ # Font kustom (jika ada)
└── README.md # Dokumentasi

## Instalasi & Deployment

1. **Hosting Lokal**:
   - Salin semua file ke folder web server (Apache, Nginx, dll.)
   - Pastikan akses ke internet tersedia untuk CDN (Tailwind CSS, Google Fonts)

2. **Hosting Online**:
   - Upload semua file ke hosting provider
   - Pastikan konfigurasi MIME types untuk file JavaScript dan CSS

3. **Static Hosting**:
   - Compatible dengan GitHub Pages, Netlify, Vercel, dll.
   - Tidak memerlukan backend khusus

## Penggunaan

1. **Login**: Masukkan password `9910` untuk mengakses sistem
2. **Pilih Vendor**: Klik tombol vendor yang sesuai dengan mesin ATM
3. **Upload Log**: Drag & drop atau paste file log ke area yang disediakan
4. **Input Data Fisik**: Masukkan data fisik remaining sesuai hasil hitung
5. **Analisis**: Klik "Filter Data" atau "Jalankan Analisa" untuk memproses
6. **AI Assistant**: Gunakan fitur AI untuk analisis lanjutan

## Spesifikasi Teknis

- **Framework**: Tailwind CSS
- **Fonts**: Google Fonts (Inter, JetBrains Mono, Mr Dafoe)
- **JavaScript**: Vanilla JavaScript (ES6+)
- **Compatibility**: Modern browsers dengan dukungan ES6
- **Responsive**: Mobile-first design

## Keamanan

- Password protection untuk akses aplikasi
- No data dikirim ke server eksternal (semua pemrosesan di client-side)
- Cache control untuk mencegah penyimpanan data sensitif

## Catatan Pengembangan

- Aplikasi ini 100% client-side, tidak memerlukan backend
- Semua kode telah dioptimalkan untuk performa
- Struktur kode modular untuk kemudahan maintenance
